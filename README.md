### 克隆工程文件
1. 在你的工程文件中使用`git clone git@gitee.com:bl_ack233/watercress-reptile.git`指令将文件克隆下载
2. 使用`npm i`下载配置文件中所有使用到的模块化工具
3. 在终端使用`node app.js`执行脚本
